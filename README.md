# Hi, I'm Zuzanna 👋

I like taking messy, complicated things and making them make sense.

My work and interests sit at the intersection of **data, product strategy, and user experience** — which means I spend a lot of time asking *why does this look like this* and then building something that explains it clearly.

My latest project is an interactive dashboard exploring patterns in the German housing market and creating insights for investors, such as identifying promising market regions and showing which features drive rental prices.

---

## What I tend to build

- End-to-end analytical workflows
- Tools and dashboards that turn analysis into something navigable
- Projects where data, structure, and UX overlap in interesting ways
